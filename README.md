# Devnullvoid Pvetui

## How do I install this cask?

First add the tap, then install the cask:

```bash
brew tap devnullvoid/pvetui
brew install --cask pvetui
```

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "devnullvoid/pvetui"
cask "pvetui"
```
